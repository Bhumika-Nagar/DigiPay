# 💳 Dummy Payment App

A full-stack dummy payment application built using the MERN stack.
This project simulates core payment features like authentication, wallet management, and secure money transfers.

---

## 🚀 Tech Stack

* **Frontend:** React
* **Backend:** Node.js, Express
* **Database:** MongoDB
* **Authentication:** JWT
* **Password Security:** bcrypt
* **Validation:** Zod

---

## ✨ Features

* 🔐 User authentication (Signup / Signin with JWT)
* 🔑 Secure password hashing using bcrypt
* 💼 Wallet system with balance management
* 💸 Send and receive money (dummy transactions)
* ⚡ Atomic balance updates using `$inc`
* 🧾 Input validation using Zod
